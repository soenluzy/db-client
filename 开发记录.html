Database Client CLI

一个支持 MySQL 和 PostgreSQL 的交互式数据库命令行工具

项目背景

由于网络限制，无法直接使用工具连接数据库，希望在 Linux 主机上部署一个 Java 程序来连接数据库执行 SQL 查询和数据导出。

配置文件示例

profiles: mysql-dev: type: mysql host: 192.168.1.100 port: 3306 database: myapp_dev username: dev_user password: xxx pgsql-dev: type: postgresql host: 192.168.1.100 port: 5432 database: myapp_dev username: dev_user password: xxx

使用示例

$ db-client --profile mysql-dev Database Client REPL Type \help for available commands [MYSQL] myapp_dev> SELECT * FROM users LIMIT 10; +----+-------+------------------+ | id | name | email | +----+-------+------------------+ | 1 | Alice | alice@example.com| | 2 | Bob | bob@example.com | +----+-------+------------------+ (2 rows in 0.023s) [MYSQL] myapp_dev> \export /tmp/users.csv ✓ Exported to /tmp/users.csv [MYSQL] myapp_dev> \help Available commands: \exit, \quit Exit the REPL \export Export last result to CSV/XLSX \help Show this help \connect Switch database connection [MYSQL] myapp_dev> \exit Goodbye!
注意：命令行输入时，反斜杠可能需要双写（如 \\export），具体取决于终端环境。

REPL 命令

命令 说明
SELECT ... 执行 SQL 查询
\export <path> 导出上次查询结果到 CSV/XLSX
\connect <profile> 切换数据库连接
\help 显示帮助信息
\exit, \quit 退出 REPL

大数据优化

针对大数据量场景，进行了以下优化：

  • 流式查询：使用 JDBC forward-only 游标，减少内存占用
  • 流式导出：Excel 使用 SXSSFWorkbook，CSV 使用批量写入
  • 进度显示：导出大数据集时显示进度百分比
